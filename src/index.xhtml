Bonjour! Bienvenue dans mon monde, le Royaume de Eric Buist! Ceci est un site Web personnel sur lequel je travaille dans mes temps libres. J'y inscris diverses informations à propos de moi-même ainsi que des expériences informatiques que j'ai vécues au cours des nombreuses années que j'ai passées à étudier dans ce domaine.

Ce site comporte deux sections principales: informations sur moi-même qui parle de moi ainsi que la section informatique qui décrit mes expériences dans ce domaine. Une page contenant des liens et une autre, des informations pour me contacter, sont aussi accessibles à partir de ce point.

Sur cette page, je propos aussi des liens vers les projets sur lesquels j'ai travaillés à l'université, que j'ai conservés sur mon ancienne page du DIRO étant donné que je ne prévois pas les mettre à jour souvent.

Il se peut que les informations trouvées sur ce site soient incomplètes ou fausses. Je m'en excuse d'avance et vous invite à me faire part de ces erreurs ou omissions pour que je les corrige. Vous pouvez me contacter pour tout commentaire en visitant la page Comment me contacter?

À noter que ce site est écrit directement en XHTML, sans passer par un éditeur HTML. J'ai utilisé le logiciel HTML Tidy pour valider et nettoyer les fichiers produits et un script Java de mon crû pour générer les barres de navigation en haut des pages.

Ce site est conçu pour un affichage dans Mozilla Firefox 1.0 ou Internet Explorer 4 ou toute version supérieure de ces navigateurs, avec une résolution minimale de 800 par 600. La plupart du contenu est en fait accessible par n'importe quel navigateur, du moment qu'il prend en charge UTF-8.

Vous souhaitez 2.25Go d'espace gratuit pour stocker vos documents, photos, etc.?

Inscrivez-vous alors à Dropbox en suivant ce lien! Après vous être enregistré sur ce site et installé le logiciel Dropbox (qui fonctionne pour Windows, Linux et Mac), tous les fichiers copiés dans le répertoire désigné sur votre ordinateur sera automatiquement sauvegardé sur le serveur et disponible sur toute autre machine sur laquelle vous avez activé Dropbox avec le même nom d'utilisateur. Dropbox permet aussi de partager des fichiers entre utilisateurs. Et c'est gratuit!
