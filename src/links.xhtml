Quelques liens et logiciels intéressants

Liens

  • Freshmeat: Ce site comporte un search engine permettant de retrouver une multitude de sites d'applications Linux, c'est l'endroit de choix pour celui qui désire obtenir un certain logiciel pour Linux.
  • Google: C'est un excellent search engine qui s'est avéré dans bien des cas plus puissant que tous els autres réunis. Fournit des résultats pertinents dans la plupart des cas.
  • Google Linux: sous-site de Google spécialisé pour des recherches portant sur Linux.
  • FreeSaver: 1400 écrans de veille gratuits pour Windows, pas de shareware ni de message de date d'expiration. Certains écrans de veille s'avèrent plutôt ordinaires tandis que d'autres comprennent d'étonnants effets 3D.
  • Webpage Backgrounds: Site contenant une foule d'arrière-plans utilisables dans des sites Web personnels
  • TweakXP: Site contenant divers trucs et astuces pour Windows XP
  • Poésie et Citations d'Amour: Collection romantique de poésie et citations d'amour.

Logiciels libres

  • Ubuntu, l'une des meilleures distributions de Linux en 2007.
  • Knoppix, une distribution Linux tenant sur un CD amorçable. Excellent pour utiliser Linux sur un ordinateur sans devoir l'y installer et pour faire des démonstrations.
  • AliXe, semblable à Knoppix mais québecois.
  • OpenMandriva, une des distributions Linux les plus abouties. S'est appelée Mandriva et Mandrake dans le passé.
  • Fedora, une autre distribution Linux semblable à Mandrake, peut-être mieux.
  • Debian, une autre distribution Linux, plus compliquée à installer mais très puissante.
  • LibreOffice.Org, une suite bureautique susceptible de remplacer Microsoft Office. S'est autrefois appelée OpenOffice, devenue LibreOffice après l'achat de Sun par Oracle.
  • Mozilla, excellent navigateur Web et logiciel de courrier électronique. Autrefois regroupé dans un seul produit, Mozilla s'est séparé en deux: Firefox pour la navigation, Thunderbird pour le courrier électronique.
  • GNU Emacs, éditeur de texte très avancé dont je ne peux toujours pas me passer.
  • The GNU Privacy Guard, logiciel de cryptographie permettant de chiffrer des messages pour en assurer la confidentialité
  • VirtualBox, émulateur permettant de faire tourner un système d'exploitation dans un autre. Très utile pour essayer Linux dans un environnement virtuel, sans risque, sans même toucher à son installation Windows. Il faut bien entendu disposer de la mémoire et de l'espace disque nécessaire pour faire tourner deux systèmes d'exploitation!
  • Docker, permet de construire des environnements virtuels légers isolés les uns des autres. En gros, un conteneur Docker est comme une machine virtuelle très légère, rapide à démarrer et à arrêter. Par contre, les conteneurs Docker ne peuvent exécuter que des variantes de Linux, pas Windows comme avec VirtualBox. Sous Windows et Mac, il faudra aussi exécuter Docker dans une machine virtuelle.
  • Pidgin, client AOL Instant Messenger pour Linux contenant aussi des plug-ins pour MSN Messenger, ICQ, etc.
  • Cygwin, ensemble de logiciels libres faisant appel à une librairie DLL fournissant une couche d'émulation Linux sous Windows. Permet de bénéficier d'outils de Linux tels que le compilateur GCC, l'utilitaire Tar pour l,archivage et même XFree86.
  • Doxygen, logiciel de génération de documentation à partir de commentaires dans les fichiers sources C/C++/Java.
  • GraphViz, logiciel permettant de tracer des graphes orientés en ne spécifiant que les sommets et les liens, aucune coordonnée absolue des sommets ne sera demandée!
  • VNC, un outil d'administration à distance. Il existe plusieurs implantations, dont TightVNC, xVNC, etc.
  • 7-zip, logiciel de compression gratuit prenant Unicode en charge
  • XMMS, bien qu'inférieur à WinAmp, c'est le lecteur MP3 Linux le plus polyvalent et le plus proche de WinAmp.
  • Audacious, clone de XMMS plus moderne
  • Audacity, logiciel d'édition audio
  • XINE, lecteur multimédia pour Linux, permet entre autres de lire des DVD et avec une version non officielle, peut lire les DVD cryptés CSS.
  • Kaffeine, lecteur multimédia utilisant Xine pour la lecture mais fournissant une interface graphique KDE beaucoup plus conviviale que Xine-UI.
  • MikTeX, implantation de TeX/LaTeX pour Windows.
  • VDMSound, émulateur de carte son Sound Blaster pour Windows NT/2000/XP, permet de refaire fonctionner de vieux jeux DOS avec son et musique!
  • DOSBox, émulateur de système DOS, pour faire tourner de vieux jeux. Probablement moins efficace que VDMSound puisque tout est émulé, pas seulement le son, mais à essayer si VDMSound seul échoue.
  • ScummVM, interpréteur pour le langage de script SCUMM utilisé par plusieurs vieux jeux d'aventures. Permet de refaire fonctionner de tels jeux dans un environnement plus moderne que le vieux moteur de jeu en mode DOS.

Logiciels non libres

Je n'utilise plus la plupart de ces logiciels de cette section puisqu'ils ne fonctionnent que sous Microsoft Windows tandis que je suis un linuxien. Mais dans le passé, ces logiciels m'ont été d'un certain secours.

  • Roxio Easy CD Creator, un logiciel de gravure de CD/DVV.
  • Nero Burning ROM, un second un autre logiciel de gravure de CD/DVD, à mon avis beaucoup plus léger et performance que Easy CD Creator.
  • Clone CD, un excellent logiciel pour la copie de CD quelconques, y compris les CD avec protections contre la copie
  • AOL Instant Messenger, un logiciel de messagerie instantannée capable de rivaliser avec MSN Messenger et ICQ
  • ICQ, le si célèbre logiciel de communication en temps réel
  • LiteStep, un remplacement pour l'Explorer de Windows parfois trop lent et trop instable. Une version française est également disponible.
  • Trillian: excellent logiciel de communication combinant MSN Messenger, ICQ et AOL Instant Messenger dans un seul programme avec une interface attrayante et toutes les fonctionnalités comme le transfert de fichiers et les messages instantanés, dommage qu'il n'existe pas pour Linux...
  • WindowBlinds, un programme qui permet de modifier plus profondément l'interface de Windows en assignant des textures aux différents éléments tels les boutons, les cases à cocher, etc. Malheureusement, le programme ralentit un peu le système, puisqu'il se charge en mémoire.
  • PowerArchiver, logiciel gratuit permettant l'archivage de la même façon que le fait WinZip
  • Partition Magic, un excellent logiciel permettant de redéfinir la structure de partitions d'un disque dur sans devoir reformater, en tout cas théoriquement. Il peut par exemple permettre de redimensionner une partition Windows de façon à faire place à Linux dans un système dual boot.
  • WinAmp, le lecteur MP3 le plus connu. Pour ceux qui, comme moi, n'aiment pas WinAmp 3 pour ses sérieuses lacunes en raccourcis claviers, sa lourdeur ou tout autre facteur, WinAmp 2 est encore disponible!
  • WinRAR, un outil de compression dont je me sers pour effectuer mes sauvegardes. Il permet de séparer de archives sur plusieurs disquettes.
  • WinZip, logiciel de compression au format ZIP, essentiel ou presque...
  • Adventure Game Studio, logiciel pour construire des jeux d'aventure semblables à King's Quest, quelque chose que j'aurais dû avoir voilà quelques années, dans le temps où j'avais envie de créer des jeux.
