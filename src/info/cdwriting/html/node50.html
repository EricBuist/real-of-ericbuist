10. Copie d'un CD

Attention!

La copie de la plupart des CD commerciaux s'avère illégale si elle n'est pas à usage privé dans le but d'effectuer une copie de sécurité.

Il existe deux méthodes principales pour parvenir au même objectif. La plus simple, qui fonctionnera dans la plupart les cas, effectue une copie physique du disque et de son contenu par la même occasion. Il est aussi possible de dupliquer un disque en régénérant sa structure logique. Ces méthodes fonctionnent bien pour les CD et DVD de données, mais dans le cas de DVD Video, d'autres facteurs doivent être considérés.
