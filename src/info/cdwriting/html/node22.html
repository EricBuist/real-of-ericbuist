4. Le graveur de CD/DVD

Après avoir étudié les différents types de disques disponibles sur le marché et les différentes méthodes d'écriture que chacun d'eux supporte, il est temps d'étudier les graveurs eux-mêmes. Plusieurs compagnies fabriquent ce genre d'appareils et nous ne pouvons dresser ici une liste exhaustive de tous les graveurs. Nous allons plutôt aborder les différents types de graveurs disponibles ainsi que les technologies importantes que ceux-ci peuvent supporter. Un graveur se caractérise par sa vitesse, son mode de connexion à l'ordinateur et le type de disques qu'il est en mesure de traiter. Il se doit également de supporter un certain nombre de technologies additionnelles afin de se distinguer de ses concurrents.
