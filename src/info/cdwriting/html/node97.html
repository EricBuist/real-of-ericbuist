B. Problème de gravure CD/DVD avec le noyau Linux 2.6.8

La version 2.6.8 du noyau Linux a rendu tous les logiciels de gravure de CD/DVD, ainsi que certains autres logiciels, inutilisables par les utilisateurs. Ce problème affecte toutes les distributions existantes du système d'exploitation à moins que les constructeurs de certaines distributions modifient le noyau pour corriger le problème. Le bogue semble là pour rester, car il provient d'un problème de sécurité existant dans les noyaux précédant 2.6.8
